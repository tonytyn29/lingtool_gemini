## 🏗️ 技术架构

### 前端技术栈
```
React 18 + TypeScript
├── UI 框架: Ant Design 5
├── 状态管理: Zustand
├── 路由: React Router 6
├── 动画: Framer Motion
├── 样式: CSS3 + 渐变色
└── 构建: Create React App 5
```

### 数据流架构
```
用户操作 → 组件 → Store → 本地存储
    ↓
AI 服务 ← 工具函数 ← 状态管理
    ↓
记忆曲线算法 → 学习建议
```

### 文件结构
```
src/
├── components/     # 通用组件
├── pages/         # 页面组件
├── stores/        # 状态管理
├── utils/         # 工具函数
├── types/         # 类型定义
├── config/        # 配置文件
└── data/          # 示例数据
```
